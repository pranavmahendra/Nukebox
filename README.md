# Nukebox
Test

Task Description:

* 5X5 blocks puzzle game.

* There will be 5 colors to connect with each other. (Red, Green, Blue, Orange, Purple)

* 5X5 grid will be having 2 nodes of each color in it when the level starts.


Objective:

* We need to connect to the same color node to each other to win the level.

Win Condition:

* Connect all nodes with their respective colors and cover the entire board to solve each puzzle (the board should not be empty) after connecting all the respective nodes to each other.

* The connection will break if they cross or overlap some other colors.
